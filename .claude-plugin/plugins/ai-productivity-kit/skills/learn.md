# ai-kit:learn

**Capture learning from session**

## Description
บันทึกสิ่งที่เรียนรู้จากการทำงาน สกัดเป็น knowledge ที่ใช้ได้

## What It Does
1. ถามว่าเรียนรู้อะไรใน session นี้
2. สกัดเป็น:
   - Trick (ถ้าเป็นเทคนิคโค้ด)
   - Pattern (ถ้าเป็นวิธีคิด/ workflow)
   - Note (ถ้าเป็นข้อมูลทั่วไป)
3. บันทึกลง ψ/memory/

## Usage
```
/ai-kit:learn
```

## Template
AI จะถาม:
```
วันนี้เรียนรู้อะไรใหม่บ้าง?

1. เทคนิคโค้ด (trick)
2. Pattern การทำงาน (pattern)
3. ข้อมูลทั่วไป (note)
```

แล้วสร้างไฟล์ตามประเภท
